argMath dapat melakukan operasi matematika sederhana dengan menggunakan argument sebagai input operator dan angka.
Ada 6 operasi matematika yang bisa dilakukan oleh argMath, yaitu:
operator + untuk penjumlahan, operator - untuk pengurangan, operator * untuk perkalian, operator / untuk pembagian dan operator \ untuk sisa bagi, operator ~ untuk pangkat.

Cara pemakaian:
Operasi Penjumlahan:
node argMath + 2 2
Akan menghasilkan nilai 4

Operasi Pengurangan:
node argMath - 3 2
Akan menghasilkan nilai 1

Operasi Perkalian:
node argMath * 3 2
Akan menghasilkan nilai 6

Operasi Pembagian:
node argMath / 3 2
Akan menghasilkan nilai 1.5

Operasi Sisa Bagi:
node argMath \ 3 2
Akan menghasilkan nilai 1

Operasi Pangkat:
node argMath ~ 3 2
Akan menghasilkan nilai 9
